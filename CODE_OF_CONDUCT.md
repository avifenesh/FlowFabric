# Code of Conduct

This project adopts the **Contributor Covenant v2.1** verbatim. The
canonical text is maintained at:

- HTML: <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>
- Markdown: <https://www.contributor-covenant.org/version/2/1/code_of_conduct/code_of_conduct.md>

By participating in this project — issues, pull requests, discussions,
review comments, commit messages, or any other project-affiliated
space — you agree to uphold that standard.

## Scope

Applies to all project spaces and to public spaces when an individual
is representing the project or its community.

## Enforcement contact

Reports of unacceptable behaviour go to the maintainer:

- **Email:** `aviarchi1994@gmail.com`
  Subject line prefix: `[FlowFabric CoC]`.
- **GitHub:** private message to [@avifenesh](https://github.com/avifenesh).

Reports are handled confidentially. The enforcement ladder and
response obligations are the ones described in the Contributor
Covenant v2.1 "Enforcement Guidelines" section (see the URLs above).
